# Euchre Card Rankings

An interactive educational tool for learning how card rankings work in the card game Euchre.

## What It Does

Select a trump suit and see how the card hierarchy shifts — including the **Right Bower** (Jack of trump) and **Left Bower** (Jack of the same-colored suit), which become the two highest cards in the trump suit.

## Tech Stack

- **Frontend**: React 18 + TypeScript, Vite, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend**: Express on Node.js (serves the app)

## Running Locally

```bash
npm install
npm run dev
```

The app runs on port 5000.
